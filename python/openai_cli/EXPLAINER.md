# OAI CLI: OpenAI API Integration & Content Scraping

## Why We Built It
To create a sophisticated archiving and backup CLI for OpenAI videos, images, and chat conversations with web dashboard, database tracking, and automated workflows. The system provides comprehensive content management with high-performance processing and intelligent organization.

## When to Use It
Use OAI CLI when you need:
- Automated scraping and downloading of OpenAI-generated content
- Metadata extraction and database storage
- Duplicate detection and conflict resolution
- Batch processing with retry logic
- Complete ChatGPT conversation export with message history
- Sora video backup with metadata tracking
- Web dashboard for content management and monitoring

## Core Features

### Content Archiving
- **Image Archiving**: Automated scraping and downloading of OpenAI-generated images
- **Video Archiving**: Sora video backup with metadata tracking and format detection
- **Chat Archiving**: Complete ChatGPT conversation export with full context
- **Metadata Extraction**: Comprehensive metadata tracking and storage

### Database Integration
- **PostgreSQL Storage**: Robust database with connection pooling
- **Full-Text Search**: Advanced search across all content types
- **Metadata Tracking**: Comprehensive metadata and session management
- **Performance Optimization**: Indexing and query optimization

### Web Dashboard
- **Real-Time Monitoring**: Beautiful interface with live statistics
- **Advanced Search**: Content type filtering and sorting
- **Responsive Design**: Mobile-friendly modern UI
- **Auto-Refreshing**: Live data updates and statistics

### Automation
- **Cron Scheduling**: Automated workflows for continuous archiving
- **Docker Ready**: Production-ready containerization
- **Health Checks**: Comprehensive monitoring and alerting
- **Backup Integration**: Automated backup and restore capabilities
